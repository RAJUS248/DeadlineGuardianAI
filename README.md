# 📌 Overview

**Deadline Guardian AI** is an AI-powered productivity and workload management platform that helps students, job seekers, professionals, and entrepreneurs organize their schedules, manage deadlines, build productive habits, and achieve their goals.

Unlike traditional to-do applications, Deadline Guardian AI acts as an intelligent productivity assistant by analyzing workload, detecting procrastination, prioritizing tasks, generating AI-powered recommendations, and automatically helping users stay focused on important work.

```bash
cd deadline-guardian-ai
```

Install dependencies

```bash
npm install
```

Run Development Server

```bash
npm run dev

```

# 🔥 Firebase Hosting

## Install Firebase CLI

```bash
npm install -g firebase-tools
```

Login

```bash
firebase login
```

Initialize Firebase

```bash
firebase init hosting
```

Choose

```
Use Existing Project
```

Select your Firebase project.

Public directory

```
dist
```

Configure as SPA

```
Yes
```

Overwrite index.html

```
No
```

---

## Build Project

```bash
npm run build
```

---

## Deploy

```bash
firebase deploy
```

## Deploy Only Hosting


# 🐞 Troubleshooting

## Build Error

```bash
rm -rf node_modules
```

or on Windows

```cmd
rmdir /s /q node_modules
del package-lock.json
npm install
```

---

## Firebase Unauthorized Domain

Go to

Firebase Console

Authentication

Settings

Authorized Domains

Add

```
yourproject.web.app

yourproject.firebaseapp.com
```

---

## Rollup Error

```bash
npm install
```

or

```bash
npm install rollup --save-dev
```

---

## Firebase Deploy Error

```bash
firebase login
firebase use --add
firebase deploy
```
